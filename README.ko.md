## 기능

- **집중할 수 있는 편집 환경** — 프레임리스 창, 커스텀 타이틀 바, 깔끔한 UI
- **리치 Markdown** — GFM(표, 취소선, 체크리스트), 수식(KaTeX), Mermaid 다이어그램, 코드 하이라이팅(Prism), 하이라이트(`==text==`)
  자세한 내용은 [Markdown 문법 지원](./tests/markdown-support.md)을 참조하세요
- **멀티 탭** — 여러 파일을 열고 즉시 전환, 자동 저장, 탭별 스크롤 위치·커서·실행 취소 기록 보존
- **소스 모드** — CodeMirror 6 기반 소스 편집, Markdown 구문 하이라이팅 지원. WYSIWYG 모드와 실행 취소 기록 공유
- **듀얼 테마** — 라이트/다크 테마를 부드럽게 전환
- **다국어 지원(i18n)** — 4개 언어(简体中文, English, 日本語, 한국어) 지원. 런타임 전환, 재시작 불필요
- **접근성(a11y)** — WAI-ARIA 패턴, 키보드 내비게이션, 포커스 관리를 모든 인터랙티브 컴포넌트에 적용
- **내보내기** — HTML 및 PDF 내보내기
- **AI 사이드바** — 컨텍스트 관리, 자동 압축, 사용량 추적이 포함된 통합 AI 채팅
- **AI 도구 호출** — AI가 도구(outline, grep, read_lines, read_section)로 능동적으로 문서를 탐색하고, 잘린 컨텍스트에 의존하지 않고 웹 콘텐츠도 가져올 수 있음
- **워크스페이스 인식 AI** — 폴더를 워크스페이스로 열면 AI가 프로젝트 수준 도구(grep, find, glob, ls, read)로 전체 코드베이스를 탐색 가능. 워크스페이스 채팅은 탭 전환 후에도 유지
- **컨텍스트 뷰** — 토큰 사용량, 컨텍스트 분석, 원본 메시지를 전용 패널에서 확인
- **선택 텍스트 AI** — 선택한 텍스트에 대한 설명, 번역, 질문
- **설정 탭** — 외관, AI 설정, 프록시, 정보를 통합한 설정 패널
- **프록시 지원** — AI 요청 및 웹 콘텐츠 가져오기에 HTTP/HTTPS/SOCKS5 프록시 사용
- **찾기 및 바꾸기** — 정규식, 대소문자 구분, 모두 바꾸기 지원
- **아웃라인 사이드바** — 문서 제목 트리, 클릭으로 이동, 활성 제목 추적
- **Mermaid 다이어그램** — 순서도, 시퀀스 다이어그램, 클래스 다이어그램 등 인라인 SVG 렌더링
- **워크스페이스 및 파일 트리** — 폴더를 워크스페이스로 열고 트리에서 파일 탐색. 우클릭으로 파일/폴더 생성, 이름 변경, 삭제
- **이미지 관리** — 붙여넣은 이미지를 `./assets/`에 자동 저장, Markdown 내 상대 경로 사용, HTML 내보내기 시 base64 임베딩
- **자동 업데이트** — 시작 시 자동 확인, 백그라운드 다운로드, 원클릭 설치 및 재시작
- **상태 표시줄** — 단어 수, 커서 위치, 파일 정보를 한눈에 확인
- **Tailwind 우선 스타일링** — 컴포넌트 스타일은 Tailwind CSS 유틸리티 클래스 + CSS 커스텀 속성으로 테마 전환 구현. CSS 파일에는 에디터 DOM 오버라이드만 남음
