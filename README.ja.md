## インストール

[Releases ページ](https://github.com/xmm1989218/moflow/releases/latest)から最新のインストーラーをダウンロードしてください。

| プラットフォーム | ダウンロード |
|---|---|
| Windows | `MoFlow_x.y.z_x64-setup.exe` |

> インストール後、MoFlow は起動時に自動的にアップデートを確認し、新バージョンが利用可能になった際にお知らせします。

## 機能

- **集中できる編集環境** — フレームレスウィンドウ、カスタムタイトルバー、シンプルな UI
- **リッチ Markdown** — GFM（テーブル、取り消し線、タスクリスト）、数式（KaTeX）、Mermaid 図、コードハイライト（Prism）、ハイライト（`==text==`）
  詳細は [Markdown 構文サポート](./tests/markdown-support.md)を参照
- **マルチタブ** — 複数ファイルを開いて瞬時に切り替え、自動保存、タブごとにスクロール位置・カーソル・アンドゥ履歴を保持
- **ソースモード** — CodeMirror 6 搭載のソース編集、Markdown シンタックスハイライト付き。WYSIWYG モードとアンドゥ履歴を共有
- **デュアルテーマ** — ライトテーマとダークテーマをスムーズに切り替え
- **多言語対応（i18n）** — 4言語（简体中文、English、日本語、한국어）をサポート。ランタイムで切り替え、再起動不要
- **アクセシビリティ（a11y）** — WAI-ARIA パターン、キーボードナビゲーション、フォーカス管理を全インタラクティブコンポーネントに適用
- **エクスポート** — HTML および PDF エクスポート
- **AI サイドバー** — コンテキスト管理、自動コンパクト、使用量トラッキング付きの統合 AI チャット
- **AI ツール呼び出し** — AI がツール（outline、grep、read_lines、read_section）で文書を自ら探索し、切り詰められたコンテキストに依存せずウェブコンテンツも取得可能
- **ワークスペース対応 AI** — フォルダをワークスペースとして開くと、AI がプロジェクトレベルのツール（grep、find、glob、ls、read）でコードベース全体を探索可能。ワークスペースチャットはタブ切り替え後も保持
- **コンテキストビュー** — トークン使用量、コンテキストの内訳、生メッセージを専用パネルで確認
- **選択テキスト AI** — 選択したテキストの解説、翻訳、質問
- **設定タブ** — 外観、AI 設定、プロキシ、バージョン情報を統合した設定パネル
- **プロキシ対応** — AI リクエストやウェブコンテンツ取得に HTTP/HTTPS/SOCKS5 プロキシを使用
- **検索と置換** — 正規表現、大文字小文字区別、全置換に対応
- **アウトラインサイドバー** — ドキュメントの見出しツリー、クリックでジャンプ、アクティブ見出しの追跡
- **Mermaid 図** — フローチャート、シーケンス図、クラス図などをインライン SVG でレンダリング
- **ワークスペース＆ファイルツリー** — フォルダをワークスペースとして開き、ツリーでファイルを閲覧。右クリックでファイルやフォルダの作成・名前変更・削除
- **画像管理** — 貼り付け画像を `./assets/` に自動保存、Markdown 内は相対パス、HTML エクスポート時に base64 埋め込み
- **自動アップデート** — 起動時にサイレントチェック、バックグラウンドダウンロード、ワンクリックでインストール＆再起動
- **ステータスバー** — 文字数、カーソル位置、ファイル情報を一目で確認
- **Tailwind ファーストのスタイリング** — コンポーネントスタイルは Tailwind CSS ユーティリティクラス + CSS カスタムプロパティでテーマ切り替えを実現。CSS ファイルに残るのはエディタ DOM 上書きのみ
